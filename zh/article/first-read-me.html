初始化 VuePress 项目

# 安装
yarn global add vuepress # 或者：npm install -g vuepress

# 新建一个 markdown 文件
echo '# Hello VuePress!' > README.md

# 开始写作
vuepress dev .

# 构建静态文件
vuepress build .

开始使用

vuePress 不同于Hexo，它是基于node，不同于wordpress，基于php，vue驱动，PWA支持，2018注定不平凡的一年，Safari，chrome 已经支持，虽然仍有一些部分特性不太完善，但是静态的搭建，已经足够， vue组件开发的特性很方便，支持markdown。然后还有一些bulabula的 就不展开了。如果你对vue技术有一定了解，相信很快上手。
